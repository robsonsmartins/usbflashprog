Detailed Description

Sivava Programmer (PCB4.5) Emulation Library.
Provides all stuff need for Sivava Programmer (PCB4.5) emulation, in a single DLL (requires I/O Wrapper Library, IO.DLL).

Compatible with the following programmers:

  • Sivava PCB4.5 at http://www.sivava.com/
  • Sivava PCB5.0 at http://www.sivava.com/
  • Willem Mod to PCB4.5 at http://www.py2bbs.qsl.br/willem_pcb45.php

Uses the following Micrososft Windows registry key: HKEY_CURRENT_USER\Software\EmuProg\
and the following registry values:

  • ChipLib [REG_SZ ]: Name of chip emulator DLL.
  • ChipSize [REG_DWORD]: Chip size, in bytes (for 8 bits devices) or in words (for 16 bits devices).
