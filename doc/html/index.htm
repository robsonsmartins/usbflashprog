Device Programmer Emulator

The Device Programmer Emulator is a C++ class library, a IO.DLL wrapper and a group of DLL's, that provides software emulation funcionalities for a lot of Device Programmers (connected in PC standard Parallel Port) and her Target Chips.

The supported plattforms:

  • Microsoft Windows 9x or NT series (32 bit).
