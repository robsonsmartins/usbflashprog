kCmdDeviceSetFlags 

OPCODE / DEVICE : Opcode Device Set Flags.

The parameter (one byte) represents the flags, and follows the table: 

+----------------------+
|Bit| Description      |
| 0 | Skip Write 0xFF  |
| 1 | Prog with VPP On |
| 2 | VPP/~OE Pin      |
| 3 | ~PGM/~CE Pin     |
| 4 | PGM positive     |
+----------------------+
kCmdDeviceSetupBus 

OPCODE / DEVICE : Opcode Device Setup Bus.

The parameter (one byte) represents the operation, and follows the table: 

+------------------------------+
|Operation| Description        |
|  0x00   | Reset Bus          |
|  0x01   | Prepare to Read    |
|  0x02   | Prepare to Program |
+------------------------------+
See also
kCmdDeviceOperationEnum
