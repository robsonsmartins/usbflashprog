Detailed Description

Willem/Sivava Programmer Emulation Library.
Provides all stuff need for Willem/Sivava Programmer emulation, in a single DLL (requires I/O Wrapper Library, IO.DLL).

Compatible with the following programmers:

  • Willem Standard 3.1, 4.0, 4.1 and 4.5 at http://web.archive.org/web/20071003005002/http://www.willem.org/
  • Sivava PCB3B at http://www.mpu51.com/eprom/eprom.html
  • EzoFlash+ 4.4 at http://www.ezoflash.com

Uses the following Micrososft Windows registry key: HKEY_CURRENT_USER\Software\EmuProg\
and the following registry values:

  • ChipLib [REG_SZ ]: Name of chip emulator DLL.
  • ChipSize [REG_DWORD]: Chip size, in bytes (for 8 bits devices) or in words (for 16 bits devices).
