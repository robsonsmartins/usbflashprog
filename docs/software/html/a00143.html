kCmdDeviceSetFlags 

OPCODE / DEVICE : Opcode Device Set Flags.

The parameter (one byte) that represents the flags follows the table: 

+----------------------+
|Bit| Description      |
| 0 | Skip Write 0xFF  |
| 1 | Prog with VPP On |
| 2 | VPP/~OE Pin      |
| 3 | ~PGM/~CE Pin     |
| 4 | PGM positive     |
+----------------------+
