Detailed Description

Willem/Sivava/EzoFlash+ Programmers Emulator Class.

Emulates the Parallel Port PC Willem Standard programmers:

  • Willem Standard 3.1, 4.0, 4.1 and 4.5
  • Sivava PCB3B
  • EzoFlash+ 4.4
