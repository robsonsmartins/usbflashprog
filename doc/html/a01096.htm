Detailed Description

IO.DLL Wrapper.
Provides all functions by IO.DLL required for I/O ports access, and forwards to a compatible programmer emulator DLL.

Uses the following Micrososft Windows registry key: HKEY_CURRENT_USER\Software\EmuProg\
and the following registry values:

  • ProgLib [REG_SZ]: Name of programmer emulator DLL.
